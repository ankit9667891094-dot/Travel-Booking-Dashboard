# AnkitBooking.com

A modern and responsive Travel Booking Dashboard website made using HTML, CSS and JavaScript.

This project includes:
- Travel booking dashboard
- Hotel and flight sections
- Search bar
- Destination gallery
- Animated booking counter
- Responsive design
- Human style comments
- Modern UI design
