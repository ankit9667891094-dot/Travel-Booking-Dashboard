4. Project Description
-----------------------
AnkitBooking.com is a modern travel booking dashboard website created using HTML, CSS and JavaScript.

This project provides a travel booking user interface where users can:
- Search destinations
- Explore hotels
- View flights
- Explore holiday packages
- View booking statistics

The website is fully responsive and beginner friendly.
